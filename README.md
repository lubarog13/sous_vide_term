# sous_vide_term

Мобильное приложение для управления умным термометром на базе Arduino. Устройство поддерживает заданную температуру воды (подходит для су-вид, приготовления йогуртов или творога) и при необходимости периодически перемешивает содержимое кастрюли.

## Возможности

- **Контроль температуры** — задание целевой температуры и отображение текущей (связь с Arduino).
- **Таймер** — длительность процесса и обратный отсчёт.
- **Перемешивание** — опциональное периодическое перемешивание кастрюли по настройке.
- **Программы** — сохранение пользовательских программ и список предустановленных сценариев приготовления.
- **Погрешность температуры** — задание погрешности температуры.
- **Состояния** — приложение задаёт и синхронизирует режимы работы с устройством.

## Стек

- [Flutter](https://flutter.dev/) — iOS и Android из одной кодовой базы.
- Локальное хранение настроек и программ: `shared_preferences` (и дальнейшее развитие по мере необходимости).

## Требования

- Flutter SDK (см. `environment` в `pubspec.yaml`).
- Подключённое устройство или эмулятор для `flutter run`.

## Запуск

```bash
cd sous_vide_term
flutter pub get
flutter run
```

## Сборка

```bash
flutter build apk    # Android
flutter build ios    # iOS (на macOS)
```

## Лицензия и публикация

Проект помечен как приватный (`publish_to: 'none'` в `pubspec.yaml`).
